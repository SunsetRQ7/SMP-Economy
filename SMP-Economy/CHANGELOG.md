# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-01-15

### Added
- Complete economy system with diamond-based currency
- Advanced banking system with interest calculation
- Full-featured auction house with bidding system
- Multi-language support (English and Persian)
- Automatic database creation and schema management
- Vault economy integration
- PlaceholderAPI support
- GUI-based interfaces for all features
- Comprehensive permission system
- Performance optimization with caching
- Security features and anti-exploit protection
- Backup and recovery system
- Update checker and auto-update functionality
- Detailed transaction logging
- Admin dashboard with player management
- Command cooldown system
- Configurable transaction fees
- Daily/weekly transfer limits
- Interest-based banking system
- Bidding system for auction house
- Item categorization in auction house
- Auction duration options (1h, 6h, 24h, 7d)
- Player data caching system
- Async database operations
- HikariCP connection pooling
- SQLite and MySQL/MariaDB support
- Automatic backup system
- Schema migration system
- Error handling and logging
- Performance metrics collection
- Debug tools for administrators

### Changed
- Complete rewrite of the economy system
- Improved database schema with proper indexing
- Enhanced security with input validation
- Better performance with async operations
- Improved GUI design with animations
- Enhanced error messages and logging
- Better configuration with validation
- Improved command system organization

### Fixed
- Memory leaks in previous versions
- Database connection issues
- Race conditions in balance updates
- Security vulnerabilities
- Performance bottlenecks
- GUI rendering issues
- Command execution problems
- Configuration loading errors

### Removed
- Legacy code from previous versions
- Unused dependencies
- Inefficient database queries

## [1.0.0] - 2024-06-01

### Added
- Initial release of SMP-Economy plugin
- Basic economy system with balance commands
- Simple database integration
- Basic command structure
- Initial configuration system
